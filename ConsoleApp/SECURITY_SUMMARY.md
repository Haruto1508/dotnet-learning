SafeVault Security Project Summary

This project demonstrates secure coding practices to protect sensitive data.

Vulnerabilities Identified

- SQL Injection
- Cross-Site Scripting (XSS)
- Weak input validation
- Unauthorized access

Fixes Applied

- Parameterized SQL queries
- Input validation rules
- HTML encoding to prevent XSS
- Role-Based Access Control (RBAC)

Testing

Security tests verified validation rules, authentication logic, and role permissions.

Copilot Assistance

Microsoft Copilot assisted in generating secure code patterns, suggesting fixes for vulnerabilities, and improving the debugging process.
